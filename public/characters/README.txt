사주야 캐릭터 이미지 폴더
========================

이 폴더에 4인방 캐릭터 이미지를 넣으세요.

파일명 규칙 (반드시 이 이름으로 저장):
  baekhalma.png    → 건물주 백할매
  doryeong.png     → 근본도령
  gumiho.png       → 구미호 선생
  sinryeong.png    → 무등산 신령님

규격:
  - 크기: 1024 x 1024px (정사각형)
  - 형식: PNG (배경 제거된 것 권장)
  - 최대 용량: 1MB 이하 권장

배경 제거 방법:
  1. remove.bg 접속 (무료)
  2. 이미지 업로드
  3. 배경 제거된 PNG 다운로드
  4. 이 폴더에 위 파일명으로 저장
